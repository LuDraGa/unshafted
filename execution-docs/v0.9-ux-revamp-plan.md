# v0.9 UX Revamp Plan

**Status:** Active
**Scope:** Popup-only UX reset

## Intent

Make the popup feel simple, action-focused, and repeatable. The main screen should stay about the current contract and the next best action. Utility work belongs in explicit top-right controls, not in the analysis scroll.

## Target IA

- Main surface: empty state, active scan, quick result, deep result, opened report.
- Top-right controls: upload, history, profile.
- History: slide-over panel with recent 5 reports, Drive refresh, open/copy/export/delete.
- Profile: compact menu with account, Drive sync, Options, clear data, sign out.
- Onboarding: Previous / Next / Skip, short action-focused copy.

## Decisions

- No dedicated report page in this pass.
- Drive sync defaults on for signed-in users.
- Drive sync preference is persisted in Supabase profile and cached locally.
- If Drive becomes enabled while a report is visible, ask before backing up that report.
- Reports may stay verbose, but report controls must stay out of the main analysis path.

## Non-Goals

- No new LLM schema.
- No new history schema.
- No billing, credits, web app, or collaboration.
- No current-page analysis.
