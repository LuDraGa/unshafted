# v0.9 Attention Redesign ‚Äî Phases 1+2

**Branch:** UX_Rework
**Driver:** user feedback that the popup feels overbearing ‚Äî too much text, too many competing CTAs, and a "scroll monster" deep-analysis view.

## Goal
Treat attention as scarce currency. Each screen has one primary action. Informational blocks default to glanceable, with progressive disclosure for depth.

## Source of truth
Designer report and bug-finder report from this session. Two critical bugs (silent Drive re-enable on `getProfile()` null; redundant deep-analysis Drive upload) were fixed in a separate session.

## Phase 3 ‚Äî Single-expand chip row + verdict compression

Driver: post-Phase-2 the popup still felt like a "scroll monster" ‚Äî five always-visible accordions stacked under the verdict, each ~40-100px even when closed. The verdict card itself carried redundant copy (rationale paraphrase + plain-English summary + meta block) duplicating the headline action.

### Popup.css ‚Äî Chip-row primitive
- [x] Add `popup-top-blockers` + `popup-top-blockers-label` (inline panel, no accordion wrapper)
- [x] Add `popup-chip-section`, `popup-chip-row` (flex-wrap)
- [x] Add `popup-chip` with `[data-active='true']` filled state, hover lift, severity-tinted count badges
- [x] Add `popup-chip-panel` (single expanded panel below chip row)

### ResultCards.tsx ‚Äî ChipRow + verdict compression
- [x] New `TopBlockersInline` component (replaces "Top blockers" accordion ‚Äî always visible, no toggle)
- [x] New `ChipRow` component: single-expand `useState<string|null>`, `role=tablist` / `role=tab` with `aria-selected` + `aria-controls`, `role=tabpanel`
- [x] New `AsksList` extracted helper
- [x] Drop `ActionSummaryGrid` entirely ‚Äî replaced by TopBlockers + ChipRow at every call site
- [x] Compress quick verdict card: remove "Why this recommendation" rationale block; remove `quick.summary` paragraph (moved into Doc info chip); fold meta into one line (`Reviewed as X ¬∑ documentType`); shrink heading `text-lg` ‚Üí `text-base`
- [x] Compress deep verdict card identically: drop `plainEnglishSummary` paragraph (moved into Doc info chip); drop "Next best action" rationale block; one-line meta
